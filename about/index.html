About

I’m Jae. I’m currently a full time .NET developer and a part time computer science student.

This site is where I blog everything that I learn related to computer science, programming, and technology so I can document things for myself and share with others.

I also use this site to share and post about my personal projects as well as any others that I’m working on or contributing to.

Most of my programming and development experience is with C# and the .NET Framework. I’ve also used and played with all kinds of other languages like C/C++, Java, VB.NET, Python, JavaScript, PHP, Scheme, Clojure, and COBOL.

I’ve done some things on the front end side of the web as well with HTML, CSS, and Bootstrap. I can build custom static websites or quickly generate them with a static site generator like Jekyll or Hugo.

I have professional experience working in IT desktop support, HR information systems, and software development. Right now I’m working as a .NET developer while earning a bachelor’s degree in CS.
