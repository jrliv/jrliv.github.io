Type conversion operators

Type conversion operators are operators that converts an object from one type to another. Here are the logical operators used in C#:

Operand Description
(type) Converts a variable from one type to another.
as Same as (type) except it returns null for an invalid conversion instead of an exception.
is Checks if an object is compatiable with another type.
typeof FIND DEFINITION

Other operators

Logical operators (also known as Boolean operators) usually take Boolean values (true or false) and return a Boolean result. Here are the logical operators used in C#. They are listed in their order of precedence to each other:

Operand Description
. Increments a number by 1
new Decrements a number by 1
() Multiplies two numbers
[] Divides a number by another number
?: Divides a number by another number to find the remainder
?? Adds two numbers
